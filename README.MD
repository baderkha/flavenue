## Flavenue
A property listing tool built to list both rentals and ready-to-sell properties. This is the backend repository, the front end repo lives somewhere else.

## Requirements

- Docker
- Go v1.20
- GNU make / Cmake
- Serverless
- HTTP CLIENT (insomnia)

## Installation/QuickStart
- `go get ./...`
- `make start-db`
- `make migrate`
- `make start`
